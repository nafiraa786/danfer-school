# Danfer School Website

A modern, premium-quality, fully responsive multi-page school website built with HTML, CSS, and JavaScript.

## Features

- **Modern Design**: High-end SaaS/product website aesthetic
- **Fully Responsive**: Mobile-first approach with perfect spacing on all devices
- **Premium UI/UX**: Cards with hover effects, smooth transitions, and visual hierarchy
- **Multi-page**: Home, Notes, Announcements, Teachers, Contact, About
- **Interactive Elements**: Mobile menu toggle, search functionality, form handling
- **Clean Code**: Well-structured HTML, CSS, and JavaScript with comments

## Pages

- **Home (index.html)**: Hero section with Elementary/High School overview and announcements preview
- **Elementary (elementary.html)**: Dedicated page for K-5 programs and information
- **High School (highschool.html)**: Dedicated page for grades 9-12 programs and information
- **Notes (notes.html)**: Study materials with filtering by Elementary/High School levels
- **Announcements (announcements.html)**: Latest school news and events
- **Teachers (teachers.html)**: Staff directory split into Elementary and High School teachers
- **Contact (contact.html)**: Contact form and school information
- **About (about.html)**: School history and mission

## Image Placeholders

The website includes comprehensive image placeholders with detailed comments:

### Homepage Images

- **Hero Image**: School building or education scene
- **Elementary School Card**: Image representing elementary education
- **High School Card**: Image representing high school education

### Section-Specific Images

- **Elementary Hero**: Photo of elementary students learning or playing
- **High School Hero**: Photo of high school students in academic setting
- **Program Images**: Icons/images for different academic programs
- **Announcement Images**: Icons for school events (reopening, curriculum updates, meetings, sports day)
- **Subject Images**: Icons/thumbnails for subjects (Mathematics, Science, History, etc.)
- **Teacher Photos**: Profile photos for all teachers (Elementary: 5 teachers, High School: 6 teachers)

All placeholders use `assets/images/placeholder.svg` - a simple SVG that displays until real images are added.

## Deployment

This website is ready for deployment on GitHub Pages:

1. Upload all files to your GitHub repository
2. Go to Settings > Pages
3. Select "Deploy from a branch" and choose "main" (or your default branch)
4. The site will be available at `https://yourusername.github.io/repository-name/`

## Customization

To customize the design:

1. Update the CSS custom properties in `styles.css` for colors, fonts, etc.
2. Replace placeholder content in HTML files
3. Add your school's logo and images
4. Modify the JavaScript for additional interactivity
